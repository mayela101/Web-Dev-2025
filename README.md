## Features
- **Home Page:** Alternating content blocks and animated swirl line connecting sections  
- **Menu Page:** Interactive menu grid with categorized items and hover effects  
- **About Page:** Team section highlighting contributors with dynamic layout  
- **Contact Page:** Simple responsive contact form styled in BYTES theme  
- **Responsive Design:** Fully optimized for desktop and mobile  
- **Animations:** Scroll and hover effects using CSS transitions and react libraries 
- **Branding:** Custom PixelFont typography, pastel color palette, and pink accent scheme
